# IPA 符号键盘 - 插件使用手册

## 插件用途

本插件提供一个交互式的国际音标（IPA）虚拟键盘，旨在帮助用户快速、准确地输入各种音标符号，无需记忆复杂的键盘编码。

## 核心功能

- **多方案支持**:
    - 通过顶部的下拉框，您可以在不同的IPA方案（如“通用IPA”、“汉语普通话”）之间切换，键盘会自动显示对应方案的符号。
- **近似检索**:
    - 在搜索框中输入拉丁字母，可以快速找到发音近似的IPA符号。例如，输入 `sh` 可以找到 `ʃ`，输入 `ng` 可以找到 `ŋ`。
- **多维特征筛选**:
    - 点击下方的复选框，可以根据音标的语音学特征（如“辅音”、“元音”、“塞音”、“鼻音”等）进行精确筛选。
- **文本暂存区**:
    - 所有点击的符号都会被输入到底部的文本框中。您可以在这里自由编辑，然后点击“复制”按钮，将最终结果安全地复制到系统剪贴板。

## 如何使用

1.  从主程序的插件菜单中启动 **“IPA 符号键盘”**。
2.  在弹出的可置顶窗口中，首先选择您需要的 **“方案”**。
3.  您可以通过**搜索框**或点击**筛选复选框**来缩小符号范围。
4.  在“符号键盘”区域，**单击**您需要的符号。
5.  符号会出现在下方的“文本暂存区”。
6.  完成编辑后，点击 **“复制”** 按钮。
7.  现在您可以将复制的音标粘贴到任何需要的地方。

## 提示

- 窗口可以被设置为“总在最前”，方便您在操作其他软件时随时调用。
- 符号的显示依赖 `Doulos SIL` 字体，主程序已内置。如遇显示问题，请确保字体正常。
